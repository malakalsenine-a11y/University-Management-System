# University Management System

This project was developed as part of the Codeline Backend Bootcamp (OOP Phase).

## Overview
The project represents a University Management System by designing an Entity Relationship Diagram (ERD) and implementing C# entity models using Object-Oriented Programming principles.

## Features
- Design a complete ERD for the university system.
- Create C# models for:
  - Student
  - Instructor
  - Department
  - Course
  - Enrollment
- Apply Data Annotations for validation and database constraints.
- Implement Navigation Properties to represent entity relationships.
- Model one-to-many and many-to-many relationships using the Enrollment entity.

## Technologies
- C#
- .NET
- Object-Oriented Programming (OOP)
- Data Annotations
- Entity Relationship Diagram (ERD)
